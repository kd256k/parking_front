# 🅿️ Fitter Parker : Know Way Home (Frontend)

> **전국 주차장 정보 안내 시스템 - Marker Studios**  \
> "주차를 딱 맞춰서 하는 사람. 집(주차장) 가는 길을 안다."

본 리포지토리는 **Fitter Parker** 프로젝트의 프론트엔드(Next.js) 애플리케이션입니다.  
카카오맵 기반으로 전국 주차장 위치를 시각화하고, 목록/대시보드/댓글/관리자 기능을 제공합니다.

---

## 🛠 Tech Stack

- Next.js 16.0.10 (App Router)
- Kakao Map SDK
- Jotai (전역 상태 관리)

---

## ✨ 주요 기능

### 지도(Map)
- 카카오맵 기반 주차장 위치 표시
- 마커 타입 3종
  - 단일 마커: 조회 항목 1건
  - 그룹 마커: 동일 좌표(또는 동일 주소)로 겹치는 경우
  - 클러스터: 근접 주차장이 많을 경우 축약 표시
- 좌측 패널: 현재 화면(뷰포트) 기준 조회된 주차장 목록
- 우측 하단: 현재 위치 표시(Geolocation)
- 우측 상단: 메뉴(페이지 이동)

### 메뉴/페이지 구성
- 로그인(로그인 중: 내 정보/로그아웃)
- 주차장 목록
- 주차장 현황 통계(대시보드)
- 가이드 보기(지도 오버레이)
- 프로그램에 대해(About)
- 관리자 전용: 사용자 목록, 댓글 관리

### 인증(Auth)
- ID/PW 로그인 및 회원가입
- OAuth2 소셜 로그인(Google/Naver/Kakao/GitHub) 연동
- Access/Refresh 만료 시 리프레시 기반의 재인증 UX 지원
  - Refresh까지 만료되면 로그인 페이지로 이동

### 목록/관리 기능
- 주차장 목록: 테이블 조회 + 복합 필터
- 주차장 상세: 상세 정보 + 댓글(별점) 조회/작성(로그인 필요)
- 사용자 목록(관리자): 필터/검색 + 상세 조회/수정
- 댓글 관리(관리자): 검색 + 다중 선택 삭제

## 🔎 필터/검색

### 주차장 목록 (필터만 제공)
- 지역구분: 전국 / 17개 행정구역
- 지역상세: 지역구분이 행정구역인 경우 구/군 선택

- 주차장유형: 전체 / 노상 / 노외 / 부설
- 요금정보: 전체 / 유료 / 무료 / 혼합

### 사용자 목록 (관리자)
- 필터: 등급(사용자/관리자), 활성화(활성/비활성)

### 댓글 목록 (관리자)
- 검색: 검색조건(내용, 주차장명, 사용자명)

## ⚙️ Environment Variables

프로젝트 루트에 `.env` 파일을 생성한 뒤 아래 값을 설정하세요.

```bash
# (필수) 브라우저에서 호출할 백엔드 주소
NEXT_PUBLIC_BACKEND_URL=http://localhost:8080

# (선택) SSR에서 사용할 내부 백엔드 주소(환경에 따라 동일하게 둬도 됨)
INTERNAL_BACKEND_URL=http://localhost:8080

# (필수) 카카오맵 JavaScript 키
NEXT_PUBLIC_KAKAO_MAP_APPKEY=[YOUR_KAKAO_APP_KEY]
```

> 참고: Geolocation(내 위치 찾기)은 운영 환경에서 HTTPS가 권장됩니다.

## 🔁 OAuth2 로그인 흐름(프론트 관점)

1. 사용자가 소셜 로그인 버튼 클릭
2. 백엔드 OAuth2 로그인 성공 후 프론트로 리다이렉트: `/login/oauth2callback`
3. 프론트 `/login/oauth2callback`에서 백엔드 API 호출: `/api/v1/auth/jwtcallback`
4. 이후 정상 로그인 상태로 화면 전환
